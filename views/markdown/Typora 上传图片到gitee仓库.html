# Typora 上传图片到gitee仓库

# 目录

  • 目录
  • 下载PicGo
  • 创建gitee仓库
  • 配置PicGo
  • 配置 Typora

# 下载PicGo

点击下载->PicGo 2.2.2

如果下载速度慢，可以使用Motrix工具进行下载

安装完成之后，配置PicGo, 下载插件 gitee-uploader 1.1.2

# 创建gitee仓库

  • 创建gitee仓库

    • 设置仓库名称： 例如: picture-bed
    • 设置开源模式为： 公开

  • 勾选使用Readme文件初始化这个仓库

    • 分支模式选择： 单分支模式（只创建master分支）

  • 创建仓库

# 配置PicGo

  • 配置PicGo

    • 设置repo: linxuxin/picture-bed
    • 设置分支: master
    • 设置token: gitee的私人token
    • 设置path：/imgs （自己设置路径）
    • 后面两个项可以不用设置，点击确定，设为默认图床

    ​

    获取token(只勾选projects即可)

    生成的token 只显示一次，如果忘记了，需要重新生成。

    # 配置 Typora

    • 配置 Typora 上传图片

      在Typora 文件菜单下选择偏好设置，如下设置

    验证的时候如果失败了，是因为服务器端口不正确导致。手动修改下即可

至此，全部已经设置完成，在写文档的时候，插入图片自动上传到gitee仓库。
